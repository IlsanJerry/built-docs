5.2.0 Release Notesedit

Also see Breaking changes in 5.0.

Enhancementsedit

Console
  • Scripts can now use proper syntax highlighting #9433
Core
  • Support for searching against tribe nodes #9132
  • Automatically select default index pattern if there is only one #9679
  • Remove "will be cached for next time" message from loading screen #9383
Discover
  • Improved consistency with the sidebar interface #7958
Saved Objects
  • Dashboards, visualizations, and saved searches can now be renamed while saving #9087
  • Improved UI when editing saved objects #9543
  • Improved UI when viewing saved objects #9535
Status
  • Status API now includes the Kibana version and build number #9195
Timelion
  • Timelion sheets can now be deleted #9191
Visualize
  • Heatmap visualization #9403
  • Line and area charts now support stepped lines #9425
  • Tilemap zoom capabilities are now determined automatically when using the default Elastic Tile Service #8630

Bug fixesedit

Core
  • The Kibana logo in the loading screen now shows properly in IE11 #9921
  • Browser-specific style overrides are now properly being handled for legacy browsers #9899
Dev Tools
  • The link to the Dev Tools app is now hidden when no developer tools are enabled #9489
Discover
  • Reduce lag experienced when expanding doc table rows #9326
Plugins
  • Calling another API route via .inject() no longer fails due to a missing socket #9332
Timelion
  • Specifying yaxis() no longer forces a minimum value of 0 #9428
  • Improved dark theme support for Timelion axis and legend labels #9422
Visualize
  • Metric visualizations now show scrollbars when the value overflows the container #9481
  • Axis custom extents now support decimal values #9426
