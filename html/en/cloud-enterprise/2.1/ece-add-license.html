Manage licenses

The use of Elastic Cloud Enterprise requires a valid license, which you can obtain from Elastic and add to your installation. These steps are not required initially to use trial versions of ECE for the duration of the trial. The trial license is valid for 30 days.

Full ECE licenses that you obtain from Elastic enable the same products, features, and support that are available to Platinum subscriptions on Elastic Cloud.

When the full ECE license for your cluster expires, Elastic automatically renews the license. Elastic automatically renews the ECE license for the lifetime of your cluster.

If you have a license from 2018 or earlier, you might receive a warning that your cluster license is about to expire. Don’t panic, it isn’t really. Elastic Cloud Enterprise manages the cluster licenses so that you don’t have to. In rare cases, such as when a cluster is overloaded, it can take longer for Elastic Cloud Enterprise to reapply the cluster license.

Download a license

To download a license from Elastic:

  1. Locate the email sent to you from Elastic that includes the link to the license.
  2. Click the link, accept the licensing agreement, and click Send.
  3. Download the ECE license.

Add a license

To add a license to your ECE installation:

  1. Log into the Cloud UI.
  2. From the Platform menu, select Settings.
  3. Click Update license and select the license file that you downloaded. License files are in the JSON format.

  4. Click Add license.

    If the operation is successful, the license is added.

Check your license expiry

To check your current license expiry date:

  1. Log into the Cloud UI.
  2. From the Platform menu, select Settings.

  3. Check the Expires row:

    In this example, the license expires in 20 days.

Request a trial extension

To request a trial license extension from Elastic:

  1. Fill in the form at https://www.elastic.co/contact. Make sure to choose Elastic Cloud Enterprise as the area of interest and state that you request a trial license extension.

    Someone from Elastic will be in touch to respond to your trial extension request.

Delete a license

To delete an existing license for your ECE installation:

  1. Log into the Cloud UI.
  2. From the Platform menu, select Settings.
  3. In the License section, click Delete license and confirm the action.
