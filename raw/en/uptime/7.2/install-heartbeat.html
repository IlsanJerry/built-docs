Step 3: Install and configure Heartbeat

Uptime requires the setup of monitors in Heartbeat. These monitors provide the data you’ll be visualizing in the Uptime UI.

See the Setup Instructions in Kibana for instructions on installing and configuring Heartbeat. Additional information is available in Configure Heartbeat.

Important considerations

  • Make sure you’re using the same major versions of Heartbeat and Kibana.
  • Index patterns tell Kibana which Elasticsearch indices you want to explore. The Uptime UI requires a heartbeat-7* index pattern. If you have configured a different index pattern, you can use field aliases to ensure data is recognized by the UI.

After you install and configure Heartbeat, the Uptime UI will automatically populate with the Heartbeat monitors.
