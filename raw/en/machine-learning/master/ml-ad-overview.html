Anomaly detectionedit

Use anomaly detection to analyze time series data by creating accurate baselines of normal behavior and identifying anomalous patterns in your data set. Data is pulled from Elasticsearch for analysis and anomaly results are displayed in Kibana dashboards. Consult Setup and security to learn more about the licence and the security privileges that are required to use anomaly detection.
