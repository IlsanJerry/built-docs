Migrating from App Search on Elastic Cloud

We recommend Elastic Cloud users defer migrating from App Search to Enterprise Search until we provide a formal migration path.

At that time, we will publish the migration procedure here.

Currently, if you create an Enterprise Search deployment from an App Search snapshot, the deployment may fail. If you’ve experienced this, revert to your existing App Search deployment, and remember to disable read-only mode if enabled.
