Enterprise Searchedit

New featuresedit

  • Metricbeat-based integration has been added to Enterprise Search, allowing for easy capture and access to monitoring data generated by the solution.
  • Elastic APM agent is now available within Enterprise Search and can be used for profiling/monitoring/troubleshooting of Enterprise Search deployments.
  • Enterprise Search Health and Stats API endpoints return a unique cluster_uuid value for each deployment (matching the cluster_uuid of the underlying Elasticsearch cluster).
  • Enterprise Search Stats API now includes the number of processed asynchronous jobs (esqueues_me.*.total_processed).
