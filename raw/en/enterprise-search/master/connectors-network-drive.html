Elastic network drive connectoredit

The Elastic network drive connector is available as a connector client for network drive data sources.

Connector clients are open code connectors that can be utilized with the Build a Connector workflow.

Any changes contributed to the connector client will eventually become part of the native connector.

Availability and prerequisitesedit

The Network Drive connector client is available in Elastic versions 8.6.0+.

Currently, this connector is available in technical preview. Features in technical preview are subject to change and are not covered by the service level agreement (SLA) of features that have reached general availability (GA).
