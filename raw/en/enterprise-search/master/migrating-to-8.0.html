Migrating to Enterprise Search 8.0edit

This section discusses the changes that you need to be aware of when migrating Enterprise Search to 8.0.

See Release notes.

Elastic Stack Security is requirededit

In previous versions it was possible to run Enterprise Search without enabling the Elastic Stack security features. This mode is no longer supported in 8.0. You must configure security for the Elastic Stack before using Enterprise Search.

Security features are enabled by default on Elastic Cloud.

Removal of the Standalone Admin UIsedit

The standalone Enterprise Search management interface has been removed. Enterprise Search can be managed in Kibana, the standard management interface for Elastic.

  • The ent_search.auth.* settings have been removed.
  • The kibana.host setting is now required.
