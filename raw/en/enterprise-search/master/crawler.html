Elastic web crawleredit

The Elastic Web Crawler programmatically discovers, extracts, and indexes content from public-facing websites and knowledge bases. Use the web crawler to transform webpage content into searchable content, and keep this content in sync with webpage changes over time. No development is required.

The web crawler is a native Elasticsearch solution— it reads and writes directly to Elasticsearch indices.

If you are looking for the App Search web crawler documentation, see App Search web crawler.
