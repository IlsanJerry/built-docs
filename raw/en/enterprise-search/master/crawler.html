Elastic web crawleredit

The Elastic Web Crawler programmatically discovers, extracts, and indexes content from public-facing websites and knowledge bases. Use the web crawler to transform webpage content into searchable content, and keep this content in sync with webpage changes over time. The web crawler is a powerful and flexible tool for ingesting content into your Elastic deployment. No development is required, so you can get up and running quickly.

The web crawler is a native Elasticsearch solution— it reads and writes directly to Elasticsearch indices. Use the full power and flexibility of Elasticsearch to search and analyze your crawled data.

If you are looking for the App Search web crawler documentation, see App Search web crawler.

To compare the web crawler with the App Search web crawler, see the reference table on this page.
