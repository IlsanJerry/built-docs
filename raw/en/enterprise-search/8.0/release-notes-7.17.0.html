Breaking changesedit

  • Changes base image for Enterprise Search Docker containers from CentOS 8 to Ubuntu 20.04 LTS.
  • Removes deprecated configuration setting slowlog.level. Deployments must remove this setting before upgrading to Enterprise Search 8.0.
