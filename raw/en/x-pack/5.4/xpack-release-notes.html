Release Notes

This section summarizes the changes in each release for all of the X-Pack components: X-Pack security, X-Pack monitoring, Watcher, X-Pack reporting, X-Pack machine learning, and X-Pack graph.

As a general rule, we strive to keep backwards compatibility between minor versions (for example, 5.x to 5.y), but there might be breaking changes between major versions (for example, 5.x to 6.y). Breaking changes are listed at the top of the release notes for each version.
