Configuring Monitoring

By default, the monitoring agent collects data from all indices every 10 seconds and stores the monitoring data locally. You can configure Monitoring to:

  • Specify the indices you want to monitor.
  • Export data to a separate monitoring cluster.
  • Configure the indices used to store monitoring data.

Advanced monitoring settings enable you to control how frequently data is collected, configure timeouts, and set the retention period for locally-stored monitoring indices. You can also adjust how monitoring data is displayed. See Monitoring Settings for the complete list of settings you can configure.
