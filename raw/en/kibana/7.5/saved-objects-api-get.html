Get object API

[experimental] This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features. Retrieve a single Kibana saved object by ID.

Request

GET /api/saved_objects/<type>/<id>

Path parameters

type
(Required, string) Valid options include visualization, dashboard, search, index-pattern, config, and timelion-sheet.
id
(Required, string) The ID of the object to retrieve.

Response code

200
Indicates a successful call.

Example

Retrieve the index pattern object with the my-pattern ID:

GET api/saved_objects/index-pattern/my-pattern

The API returns the following:

{
  "id": "my-pattern",
  "type": "index-pattern",
  "version": 1,
  "attributes": {
    "title": "my-pattern-*"
  }
}
