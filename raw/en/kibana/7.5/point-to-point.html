Point to pointedit

A point-to-point connection plots aggregated data paths between the source and the destination. Thicker, darker lines symbolize more connections between a source and destination, and thinner, lighter lines symbolize less connections.

Point to point uses an Elasticsearch terms aggregation to group your documents by destination. Then, a nested GeoTile grid aggregation groups sources for each destination into grids. A line connects each source grid centroid to each destination.
