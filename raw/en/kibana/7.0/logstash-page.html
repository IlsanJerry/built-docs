Logstash Monitoring Metricsedit

If you are monitoring Logstash nodes, click Overview in the Logstash section of the Monitoring page in Kibana. You can view the overall health of the Logstash nodes.

To view Logstash node metrics, click Nodes. The Nodes section shows the status of each Logstash node.

Click the name of a node to view its statistics over time.

For more information, see Monitoring Logstash.
