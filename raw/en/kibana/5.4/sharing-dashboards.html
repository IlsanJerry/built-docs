Sharing a Dashboardedit

You can either share a direct link to a Kibana dashboard with another user, or embed the dashboard in a web page. Users must have Kibana access to view embedded dashboards.

To share a dashboard:

  1. Click Dashboard in the side navigation.
  2. Open the dashboard you want to share.
  3. Click Share.

  4. Copy the link you want to share or the iframe you want to embed. You can share the live dashboard or a static snapshot of the current point in time.

    When sharing a link to a dashboard snapshot, use the Short URL. Snapshot URLs are long and can be problematic for Internet Explorer users and other tools.
