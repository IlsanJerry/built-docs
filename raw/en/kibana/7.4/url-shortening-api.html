URL shortening APIedit

Kibana URLs contain the state of the application, which makes them long and cumbersome. Internet Explorer has URL length restrictions, and some wiki and markup parsers don’t do well with the full-length version of the Kibana URL.

Short URLs are designed to make sharing Kibana URLs easier.
