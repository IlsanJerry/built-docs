Monitoring

The X-Pack monitoring components enable you to easily monitor the Elastic Stack from Kibana. You can view health and performance data for Elasticsearch, Logstash, and Kibana in real time, as well as analyze past performance. For more information, see Monitoring the Elastic Stack.

When you install X-Pack, a monitoring agent runs on each Elasticsearch node, Logstash node, and Kibana instance to collect and index metrics. Each node and instance is considered unique based on its persistent UUID, which is written to the path.data directory when the node or instance starts.
