Kibana 8.0.1edit

Coming in 8.0.1.

The 8.0.1 release includes the following enhancement and bug fixes.

Enhancementsedit

Elastic Security
For the Elastic Security 8.0.1 release information, refer to Elastic Security Solution Release Notes.

Bug fixesedit

Alerting
Fixes an issue where the Cases rule link was displayed as Unknown rule, even when the correct rule name was retrieved #123883
Dashboard
Transfer state when drilldown is opened in a new tab #124770
Elastic Security
For the Elastic Security 8.0.1 release information, refer to Elastic Security Solution Release Notes.
Fleet
  • Avoid breaking setup when compatible package is unavailable in registry #125525
  • Tell users when assets are installed in a different space #125278
Machine Learning
Only fetch field statistics for available fields #123003
Management
Registering a snapshot repository is now possible in Snapshot and Restore #125656
Maps
  • Fixes an issue where dashboards with by-value maps are broken when copied to new space #125599
  • Fixes an issue where styles were unable to be applied to vector tiles #124289
Metrics
Allow single values for ip and mac node metadata #124384
Monitoring
Update rules queries to support Metricbeat 8.0.0 #125748
Platform
Adds origin conflict checks when resolving import errors #125241
Security
Adds 8.0.1 rules updates #125316
Uptime
Update mobile data view to account for metrics events #125916
