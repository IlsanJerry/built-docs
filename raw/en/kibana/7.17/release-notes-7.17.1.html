Kibana 7.17.1edit

The 7.17.1 release includes the following bug fixes.

Bug fixesedit

Elastic Security
For the Elastic Security 7.17.1 release information, refer to Elastic Security Solution Release Notes.
Lens & Visualizations
  • Removes tabify performance issue #124931
  • Fixes an issue where shard failures were not reported in TSVB #123474
Machine Learning
  • Fixes anomaly detector jobs list filter delay #124328
Management
  • You can now use the search bar in the snapshots list of the Snapshot & Restore app with date strings #125234
  • Adds warning when remote clusters are configured #125138
  • Updates logic for handling reindex failures #124571
  • Fixes redirect to the wizard #124337
  • Fixes cross cluster search when using the fields search request parameter for versions of Elasticsearch prior to 7.0 #123267
Maps
Fixes an issue where layers from WMS capabilities for a specific URL were unable to load #123386
Security
  • Prevents repeated calls to extend session when Kibana is unavailable #125973
  • Changed deprecation warning for elasticsearch.username #124717
