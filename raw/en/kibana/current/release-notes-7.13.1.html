Kibana 7.13.1edit

The 7.13.1 release includes the following bug fixes.

Bug Fixesedit

Alerting
  • Adds ignore_above to alerts params mappings to handle immense params #100726
Dashboard
  • Moves the by value migrations for 7.13 #100622
  • Fixes JSON editor height in Inspector on Safari #99032
Fleet
  • Adds missing install button for integrations that aren’t installed yet #100370
Lens & Visualizations
  • Fixes an issue where upgrading from 7.12.1 to 7.13.0 breaks TSVB vizualizations with no timefield defined #100864
Machine Learning
  • Fixes field used in remote desktop protocol (RDP) anomaly detection job to event.type #100000
Management
  • Fixes large integer formatting with decimal part using numeral-js #99609
Querying & Filtering
  • Using a range query in KQL against a scripted number field now works properly #99554
Sharing
  • Adds appropriate table caption for table listing generated reports #100118
Security
  • Gracefully handle malformed index patterns on role management pages #99918
