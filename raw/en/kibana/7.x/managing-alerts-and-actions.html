Rules and Connectorsedit

The Rules and Connectors UI lets you see and control all the rules in a space, and provides tools to create and manage connectors so that rules can trigger actions like notification, indexing, and ticketing.

To manage rules and connectors, open the main menu, then click Stack Management > Alerts and Insights > Rules and Connectors.

Similar to dashboards, rules and connectors reside in a space. The Rules and Connectors UI only shows rules and connectors for the current space.

Elasticsearch also offers alerting capabilities through Watcher, which can be managed through the Watcher UI. See Differences from Watcher for more information.

Required permissionsedit

Access to rules and connectors is granted based on your privileges to alerting-enabled features. See Alerting Security for more information.
