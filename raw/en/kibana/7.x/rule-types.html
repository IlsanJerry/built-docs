Rulesedit

Kibana provides two types of rules:

  • Stack rules, which are built into Kibana
  • Domain-specific rules, which are registered by Kibana apps.

Standard stack rulesedit

Kibana provides two stack rules:

  • Index threshold
  • Elasticsearch query

Users require the all privilege to access the Stack Rules feature and create and edit rules. See feature privileges for more information.

Domain-specific rulesedit

For domain-specific rules, refer to the documentation for that app. Kibana supports these rules:

  • Observability rules
  • Security rules
  • Maps rules
  • Machine learning rules [beta] This functionality is in beta and is subject to change. The design and code is less mature than official GA features and is being provided as-is with no warranties. Beta features are not subject to the support SLA of official GA features.

Some rule types are subscription features, while others are free features. For a comparison of the Elastic subscription levels, see the subscription page.
