Kibana 6.8.1edit

Before you migrate your application from one version of Kibana to another, review the breaking changes.

Bug fixesedit

Monitoring
  • Fixes the inability to use GET /api/stats API when you use query string parameters #36986
  • Fixes the inability to view the Completed recoveries on the Elasticsearch overview page #38543
  • Removes the use of the terminate_after flag for monitoring queries due to overuse #37099
  • Updates the Shard Activity table to display 60 seconds instead of 1 minute #38065
Machine Learning
  • Fixes the constant polling to deleting job tasks #38013
