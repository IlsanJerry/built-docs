Kibana breaking changesedit

This list summarizes the most important breaking changes in Kibana 8.1.0. For the complete list, go to Kibana release notes.

Removes legacy CSV export type

Details
The /api/reporting/generate/csv endpoint has been removed. For more information, refer to #121435.

Impact
If you are using 7.13.0 and earlier, regenerate the POST URLs that you use to automatatically generate CSV reports.

Removes legacy PDF shim

Details
The POST URLs that you generated in Kibana 6.2.0 no longer work. For more information, refer to #121369.

Impact
Regenerate the POST URLs that you use to automatatically generate PDF reports.
