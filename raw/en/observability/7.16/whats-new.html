APM failed transactions correlation charts and top values (Beta)edit

In 7.16, the failed transaction correlations include a new chart, which shows the transactions in the overall latency distribution for the transaction group. Both the latency and failed transaction correlation tables also have a new option to view and add filters for the most popular field values. To learn more, see Correlations.
