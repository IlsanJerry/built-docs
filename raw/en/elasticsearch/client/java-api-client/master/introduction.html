Introductionedit

This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features.

This is the official Java API client for Elasticsearch. The client provides strongly typed requests and responses for all Elasticsearch APIs. It delegates protocol handling to an http client such as the Java Low Level REST Client that takes care of all transport-level concerns.
