allocation_typeedit

This setting is used only when using the allocation action or the allocated filtertype.

The value of this setting must be one of require, include, or exclude.

Read more about these settings at http://www.elastic.co/guide/en/elasticsearch/reference/5.6/shard-allocation-filtering.html

The default value for this setting is require.
