allocatededit

- filtertype: allocated
  key:
  value:
  allocation_type:
  exclude: True

Empty values and commented lines will result in the default value, if any, being selected. If a setting is set, but not used by a given filtertype, it will be ignored.

This filtertype will iterate over the actionable list and match indices based on their shard routing allocation settings. They will remain in, or be removed from the actionable list based on the value of exclude.

Required settingsedit

  • key (required)
  • value (required)

Optional settingsedit

  • allocation_type (has a default value which can optionally be changed)
  • exclude (default is True)
