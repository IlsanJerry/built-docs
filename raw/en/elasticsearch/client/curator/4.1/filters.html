Filtersedit

Filters are the way to select only the indices (or snapshots) you want.

The index filtertypes are:

  • alias
  • age
  • count
  • pattern
  • space
  • kibana
  • forcemerged
  • closed
  • opened
  • allocated
  • none

The snapshot filtertypes are:

  • age
  • count
  • pattern
  • state
  • none

Starting in Curator 4.1, you can use environment variables in your configuration files.
