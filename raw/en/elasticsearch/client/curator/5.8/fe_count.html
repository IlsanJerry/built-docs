countedit

This setting is only used with the count filtertype and is a required setting.

- filtertype: count
  count: 10

The value for this setting is a number of indices or snapshots to match.

Items will remain in the actionable list depending on the value of exclude, and reverse.

There is no default value. This setting must be set by the user or an exception will be raised, and execution will halt.
