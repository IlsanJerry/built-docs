stats_resultedit

This setting is only used with the age filtertype.

 - filtertype: age
   source: field_stats
   direction: older
   unit: days
   unit_count: 3
   field: '@timestamp'
   stats_result: min_value

The value for this setting can be either min_value or max_value. This setting is only used when source is field_stats, and determines whether Curator will use the minimum or maximum value of field for time calculations.

The default value for this setting is min_value.
