Subcommandsedit

The Click API is used to form the command-line for Curator. Click allows Curator to have nested commands. In this way, a command-line structure can be built which allows multiple commands to all make use of the same subcommands.

Available subcommands are indices for index selection, and snapshots for snapshot selection.
