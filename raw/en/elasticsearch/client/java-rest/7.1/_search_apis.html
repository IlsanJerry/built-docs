Search APIsedit

The Java High Level REST Client supports the following Search APIs:

  • Search API
  • Search Scroll API
  • Clear Scroll API
  • Search Template API
  • Multi-Search-Template API
  • Multi-Search API
  • Field Capabilities API
  • Ranking Evaluation API
  • Explain API
  • Count API
