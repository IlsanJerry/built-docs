Document APIsedit

The Java High Level REST Client supports the following Document APIs:

Single document APIs
  • Index API
  • Get API
  • Exists API
  • Delete API
  • Update API
  • Term Vectors API
Multi-document APIs
  • Bulk API
  • Multi-Get API
  • Reindex API
  • Update By Query API
  • Delete By Query API
  • Rethrottle API
  • Multi Term Vectors API
