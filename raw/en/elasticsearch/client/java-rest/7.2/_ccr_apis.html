CCR APIsedit

The Java High Level REST Client supports the following CCR APIs:

  • Put Follow API
  • Pause Follow API
  • Resume Follow API
  • Unfollow API
  • Forget Follower API
  • Put Auto Follow Pattern API
  • Delete Auto Follow Pattern API
  • Get Auto Follow Pattern API
  • Get CCR Stats API
  • Get Follow Stats API
  • Get Follow Info API
