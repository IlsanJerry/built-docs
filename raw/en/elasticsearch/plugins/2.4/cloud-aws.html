AWS Cloud Plugin

The Amazon Web Service (AWS) Cloud plugin uses the AWS API for unicast discovery, and adds support for using S3 as a repository for Snapshot/Restore.

If you are looking for a hosted solution of Elasticsearch on AWS, please visit http://www.elastic.co/cloud.

Installation

This plugin can be installed using the plugin manager:

sudo bin/plugin install cloud-aws

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

This plugin can be downloaded for offline install from https://download.elastic.co/elasticsearch/release/org/elasticsearch/plugin/cloud-aws/2.4.6/cloud-aws-2.4.6.zip.

Removal

The plugin can be removed with the following command:

sudo bin/plugin remove cloud-aws

The node must be stopped before removing the plugin.
