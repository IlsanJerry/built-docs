GCE Cloud Pluginedit

The Google Compute Engine Cloud plugin uses the GCE API for unicast discovery.

Installationedit

This plugin can be installed using the plugin manager:

sudo bin/plugin install cloud-gce

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

This plugin can be downloaded for offline install from https://download.elastic.co/elasticsearch/release/org/elasticsearch/plugin/cloud-gce/2.4.6/cloud-gce-2.4.6.zip.

Removaledit

The plugin can be removed with the following command:

sudo bin/plugin remove cloud-gce

The node must be stopped before removing the plugin.
