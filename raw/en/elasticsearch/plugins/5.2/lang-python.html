Python Language Pluginedit

Deprecated in 5.0.0.

Python will be replaced by the new default scripting language Painless

The Python language plugin enables the use of Python in Elasticsearch scripts, via the Jython Java implementation of Python.

Installationedit

This plugin can be installed using the plugin manager:

sudo bin/elasticsearch-plugin install lang-python

The plugin must be installed on every node in the cluster, and each node must be restarted after installation.

This plugin can be downloaded for offline install from https://artifacts.elastic.co/downloads/elasticsearch-plugins/lang-python/lang-python-5.2.2.zip.

Removaledit

The plugin can be removed with the following command:

sudo bin/elasticsearch-plugin remove lang-python

The node must be stopped before removing the plugin.
