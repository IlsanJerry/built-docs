SHOW TABLESedit

Synopsis.  

SHOW TABLES [ LIKE? pattern? ]?

SQL match pattern

Description. List the tables available to the current user and their type.

SHOW TABLES;

     name      |     type
---------------+---------------
emp            |BASE TABLE
employees      |ALIAS
library        |BASE TABLE

The LIKE clause can be used to restrict the list of names to the given pattern.

The pattern can be an exact match:

SHOW TABLES LIKE 'emp';

     name      |     type
---------------+---------------
emp            |BASE TABLE

Multiple chars:

SHOW TABLES LIKE 'emp%';

     name      |     type
---------------+---------------
emp            |BASE TABLE
employees      |ALIAS

A single char:

SHOW TABLES LIKE 'em_';

     name      |     type
---------------+---------------
emp            |BASE TABLE

Or a mixture of single and multiple chars:

SHOW TABLES LIKE '%em_';

     name      |     type
---------------+---------------
emp            |BASE TABLE
