Get watch APIedit

This API retrieves a watch by its ID.

Requestedit

GET _xpack/watcher/watch/<watch_id>

Path Parametersedit

watch_id (required)
(string) Identifier for the watch.

Authorizationedit

You must have manage_watcher or monitor_watcher cluster privileges to use this API. For more information, see Security Privileges.

Examplesedit

The following example gets a watch with my-watch id:

GET _xpack/watcher/watch/my_watch
