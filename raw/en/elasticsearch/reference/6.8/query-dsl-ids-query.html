Ids Queryedit

Returns documents based on their IDs. This query uses document IDs stored in the _id field.

Example requestedit

GET /_search
{
    "query": {
        "ids" : {
            "type" : "_doc",
            "values" : ["1", "4", "100"]
        }
    }
}

Top-level parameters for idsedit

values
An array of document IDs.
