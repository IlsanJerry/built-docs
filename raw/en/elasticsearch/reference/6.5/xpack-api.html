X-Pack APIsedit

X-Pack exposes REST APIs that are used by the UI components and can be called directly to configure and access X-Pack features.

  • Info API
  • Cross-cluster replication APIs
  • Graph Explore API
  • Licensing APIs
  • Machine Learning APIs
  • Security APIs
  • Watcher APIs
  • Rollup APIs
  • Migration APIs
