Delete index template APIedit

This documentation is about legacy index templates, which are deprecated and will be replaced by the composable templates introduced in Elasticsearch 7.8. For information about composable templates, Index templates .

Deletes a legacy index template.

DELETE /_template/my-legacy-index-template

Requestedit

DELETE /_template/<legacy-index-template>

Path parametersedit

<legacy-index-template>
(Required, string) Comma-separated list of legacy index templates to delete. Wildcard (*) expressions are supported.

Query parametersedit

master_timeout
(Optional, time units) Specifies the period of time to wait for a connection to the master node. If no response is received before the timeout expires, the request fails and returns an error. Defaults to 30s.
timeout
(Optional, time units) Specifies the period of time to wait for a response. If no response is received before the timeout expires, the request fails and returns an error. Defaults to 30s.
