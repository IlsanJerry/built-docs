Delete filter API

Deletes a filter.

Request

DELETE _xpack/ml/filters/<filter_id>

Description

This API deletes a filter. If a machine learning job references the filter, you cannot delete the filter. You must update or delete the job before you can delete the filter.

Path Parameters

filter_id (required)
(string) Identifier for the filter.

Authorization

You must have manage_ml, or manage cluster privileges to use this API. For more information, see Security privileges.

Examples

The following example deletes the safe_domains filter:

DELETE _xpack/ml/filters/safe_domains

When the filter is deleted, you receive the following results:

{
  "acknowledged": true
}
