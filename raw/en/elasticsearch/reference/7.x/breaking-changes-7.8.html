Breaking changes in 7.8edit

This section discusses the changes that you need to be aware of when migrating your application to Elasticsearch 7.8.

See also Release highlights and Release notes.

Coming in 7.8.0.

Settings changesedit

node.local_storage is deprecatededit

In Elasticsearch 7.8.0, the setting node.local_storage was deprecated and beginning in Elasticsearch 8.0.0 all nodes will require local storage.
