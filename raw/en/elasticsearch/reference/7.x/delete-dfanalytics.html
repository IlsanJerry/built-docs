Delete data frame analytics jobs APIedit

Deletes an existing data frame analytics job.

This functionality is experimental and may be changed or removed completely in a future release. Elastic will take a best effort approach to fix any issues, but experimental features are not subject to the support SLA of official GA features.

Requestedit

DELETE _ml/data_frame/analytics/<data_frame_analytics_id>

Prerequisitesedit

  • You must have machine_learning_admin built-in role to use this API. For more information, see Security privileges and Built-in roles.

Path parametersedit

<data_frame_analytics_id>
(Required, string) Identifier for the data frame analytics job you want to delete.

Examplesedit

The following example deletes the loganalytics data frame analytics job:

DELETE _ml/data_frame/analytics/loganalytics

The API returns the following result:

{
  "acknowledged" : true
}
