Configuring monitoring in Elasticsearchedit

If you enable the Elastic monitoring features in your cluster, there are two methods to collect metrics about Elasticsearch:

  • Collecting monitoring data
  • Collecting monitoring data with Metricbeat

You can also use Filebeat to collect Elasticsearch logs.

To learn about monitoring in general, see Monitoring the Elastic Stack.
