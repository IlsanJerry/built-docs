Or Queryedit

Deprecated in 2.0.0-beta1.

Use the bool query instead

A query that matches documents using the OR boolean operator on other queries.

{
    "filtered" : {
        "query" : {
            "term" : { "name.first" : "shay" }
        },
        "filter" : {
            "or" : [
                {
                    "term" : { "name.second" : "banon" }
                },
                {
                    "term" : { "name.nick" : "kimchy" }
                }
            ]
        }
    }
}
