Full cluster restart upgradeedit

When upgrading to Elasticsearch 8.0 and later, you must first upgrade to 7.17 even if you opt to perform a full-cluster restart instead of a rolling upgrade. For more information about upgrading, see Upgrading to Elastic 8.0.0-rc2.

role="exclude",id="rolling-upgrade"] === Rolling upgrade

When upgrading to Elasticsearch 8.0 and later, you must first upgrade to 7.17 whether you opt to perform a rolling upgrade (upgrade one node at a time) or a full-cluster restart upgrade. For more information about upgrading, see Upgrading to Elastic 8.0.0-rc2.
