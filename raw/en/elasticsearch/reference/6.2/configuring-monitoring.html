Configuring Monitoring in Elasticsearch

X-Pack monitoring is enabled by default when you install X-Pack. Advanced monitoring settings enable you to control how frequently data is collected, configure timeouts, and set the retention period for locally-stored monitoring indices. You can also adjust how monitoring data is displayed. For more information, see Monitoring Elasticsearch.
