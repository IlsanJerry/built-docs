Retrieving a runtime fieldedit

Use the fields parameter on the _search API to retrieve the values of runtime fields. Runtime fields won’t display in _source, but the fields API works for all fields, even those that were not sent as part of the original _source.

The following request uses the search API to retrieve the day_of_week field that this previous request defined as a runtime field in the mapping. The value for the day_of_week field is calculated dynamically at search time based on the evaluation of the defined script.

GET my-index/_search
{
  "fields": [
    "@timestamp",
    "day_of_week"
  ],
  "_source": false
}
