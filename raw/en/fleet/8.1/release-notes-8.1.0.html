Fleet and Elastic Agent 8.1.0edit

Review important information about the Fleet and Elastic Agent 8.1.0 release.

New featuresedit

The 8.1.0 release adds the following new and notable features.

Elastic Agent
  • Add support for loading input configurations from external configuration files in standalone mode. You can load inputs from YAML configuration files under the folder {path.config}/inputs.d. #30087

Enhancementsedit

Fleet
  • Add shipper label. #122491
  • Add support for non-superuser access to Fleet and Integrations. #122347
  • Add support for bundling packages as zip archives with Kibana source. #122297
  • Make the default integration install explicit. #121628

Bug fixesedit

Fleet
  • Re-add missing packages to keep up to date list. #125787
  • Trim whitespace from package policy names. #125400
Elastic Agent
  • Fix agent configuration overwritten by default Fleet config. #29297
