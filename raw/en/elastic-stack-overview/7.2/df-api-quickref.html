API quick referenceedit

All data frame transform endpoints have the following base:

/_data_frame/transforms/
  • Create data frame transforms
  • Delete data frame transforms
  • Get data frame transforms
  • Get data frame transforms statistics
  • Preview data frame transforms
  • Start data frame transforms
  • Stop data frame transforms

For the full list, see Data frame transform APIs.
