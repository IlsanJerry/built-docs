Logstash X-Pack 6.2.4 Release Notes

There are enhancements and bug fixes for the Pipeline Viewer UI and the pipeline management feature in Kibana. For more information, see Kibana X-Pack 6.2.4 Release Notes.

Bug fixes

  • Fixes a bug in the ArcSight module when using the Kafka input. The CEF codec was misconfigured to expect new-line delimited messages instead of discrete ones. This bug caused the kafka input to buffer messages indefinitely.
