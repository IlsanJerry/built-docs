aws rds metricsetedit

The rds metricset of aws module allows you to monitor your AWS RDS service. rds metricset fetches a set of metrics from Amazon RDS and Amazon Aurora DB. with Amazon RDS, users can monitor network throughput, I/O for read, write, and/or metadata operations, client connections, and burst credit balances for their DB instances. Amazon RDS sends metrics and dimensions to Amazon CloudWatch every minute. Amazon Aurora provides a variety of Amazon CloudWatch metrics that users can use to monitor health and performance of their Aurora DB cluster.

AWS Permissionsedit

Some specific AWS permissions are required for IAM user to collect AWS RDS metrics.

cloudwatch:GetMetricData
ec2:DescribeRegions
rds:DescribeDBInstances
sts:GetCallerIdentity
iam:ListAccountAliases

Dashboardedit

The aws rds metricset comes with a predefined dashboard.

Configuration exampleedit

- module: aws
  period: 60s
  metricsets:
    - rds
  access_key_id: '<access_key_id>'
  secret_access_key: '<secret_access_key>'
  session_token: '<session_token>'
