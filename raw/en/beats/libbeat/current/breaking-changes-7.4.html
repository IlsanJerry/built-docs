Breaking changes in 7.4edit

This section discusses the main changes that you need to be aware of to migrate Beats to version 7.4. See the release notes for a complete list of breaking changes, including changes to beta or experimental functionality.

Golang version updateedit

The Beats platform has been updated to use Golang 1.12.9.

Auditbeat system module changesedit

The socket dataset in the system module now uses Kprobes for finer-grained monitoring and UDP support. For more information, see System socket dataset.

Filebeat field name changesedit

Some field names exported by the asa fileset in the cisco module have changed:

Old fieldNew field

log.original

event.original

cisco.asa.list_id

cisco.asa.rule_name
