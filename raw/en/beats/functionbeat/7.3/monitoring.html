Monitoring Functionbeat

You can use the Elastic Stack monitoring features to gain insight into the health of Functionbeat agents running in your environment.

To monitor Functionbeat, make sure monitoring is enabled on your Elasticsearch cluster, then configure the method used to collect Functionbeat metrics. You can use the following method:

  • Internal collection

To learn about monitoring in general, see Monitoring the Elastic Stack.
