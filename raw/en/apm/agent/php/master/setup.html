Set up the Agentedit

Set up the Agent by compiling and installing this library as a PHP extension. It is currently only available for a unix-lke operating system.

The agent supports PHP versions 7.2-7.4.

Execute the following commands to install:

cd src/ext
phpize
CFLAGS="-std=gnu99" ./configure --enable-elastic_apm
make clean
make
sudo make install

Enable the extension by adding the following to your php.ini file:

extension=elastic_apm.so
elastic_apm.bootstrap_php_part_file=<repo root>/src/bootstrap_php_part.php
