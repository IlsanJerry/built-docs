APM Server version 7.2.0edit

Addededit

  • Make stacktrace lineno optional 2105.
  • Return response body with number of accepted events when queries pass a verbose param. 2110.
  • Add Idle timeout to server config 2122.
  • Upgrade Go to 1.12.4 2132.
  • Add geoip processing to the default ingest pipeline 2177.
  • Add ephemeral_id attribute in the metadata 2179.
  • Add ILM support for APM Server, using fixed policies 2099, 2209.
  • Add setup --index-management cmd, deprecate setup --template cmd 2180, 2099.
  • Support more SSL config options for agent/server communication 2224, 2281.
