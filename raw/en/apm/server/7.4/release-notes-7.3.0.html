Bug fixesedit

  • Initial support for remote agent configuration, requires Kibana 2289, 2301, 2386, 2407, 2421, 2441, 2443.
  • Add basic caching to remote agent configuration 2337.
  • Enable APM pipeline by default 2301.
  • Add fields required by breakdown graphs APM pipeline by default 2315, 2397.
  • Require client auth when CA is configured and SSL enabled 2340.
  • Enable ILM by default when configured Elasticsearch >= 7.0 2356.
