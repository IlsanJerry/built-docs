Installing APM Serveredit

Download APM Server for your operating system and extract the package. Then follow the instructions on Setting up and running APM Server.

You can also install APM Server from our repositories, on macOS with Homebrew, or as a service on Windows:

  • Repositories for APT and YUM
  • Install APM Server on macOS with Homebrew
  • Installing APM Server on Windows

To run APM Server in Docker, see Running APM Server on Docker.

To run APM Server under systemd, see APM Server and systemd.
