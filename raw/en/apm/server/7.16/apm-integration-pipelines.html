Pipelinesedit

The APM integration loads default ingest node pipelines into Elasticsearch. These pipelines preprocess and enrich APM documents before indexing them. To view and edit these pipelines in Kibana, select Stack Management / Index Node Pipelines. Search for apm.

See ingest node pipelines for more information.
