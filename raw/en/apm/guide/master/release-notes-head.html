APM version HEADedit

View commits

Starting in version 8.0.0, Fleet uses the APM integration to set up and manage APM index templates, ILM policies, and ingest pipelines. APM Server will only send data to Elasticsearch after the APM integration has been installed. See Step 2: Set up and configure for more information.

Breaking Changesedit
