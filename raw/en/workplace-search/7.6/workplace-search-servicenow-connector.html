ServiceNow connectoredit

Ingest all of your ServiceNow files into Workplace Search.

The ServiceNow Connector requires that you create an OAuth App within ServiceNow.

Configurationedit

Login to ServiceNow.

Access your administrative dashboard.

Use the search bar on the left-hand side to search for Application Registry.

Give it a click:

Figure 65. Connecting ServiceNow

We will need a new application.

Click the New button:

Figure 66. Connecting ServiceNow

A trio of text links appear.

Select Create an OAuth API endpoint for external clients:

Figure 67. Connecting ServiceNow

Next, provide a name for the application. Elastic feels good. Oooh yeah.

And then you’ll also want to provide a redirect URL:

  • http://localhost:3002/ws/org/sources/service_now/create

Remember: the hostname should be relative to the one you have configured.

After that, click Submit:

Figure 68. Connecting ServiceNow

The app is created, but now we need to retrieve the Client ID and Client Secret.

Find your app within the list and re-enter it.

Retrieve the values!

Figure 69. Connecting ServiceNow

That’s it for ServiceNow.

Enter these values within Workplace Search!

Congratulations! It’s time to search.

You can now make the source available to various groups and scale its relevance priority.
