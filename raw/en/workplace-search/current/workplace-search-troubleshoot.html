Troubleshooting Workplace Search

Discovered an issue, a bug or have a suggestion to improve the experience? Don’t hesitate to send any report to our team.

There is no migration path between beta versions; you must download the latest version and set up the experience as new.

Elasticsearch Cluster Settings

Workplace Search makes alterations to the auto_create_index persist and transient settings within Elasticsearch.

Avoid making changes to those setting as doing so will create issues.

Read the Elasticsearch reference docs for more information.

Password Reset Tips

Run the binary with an attached --reset-auth command:

  • Standard Auth: Resets login and generates new random password.
  • elasticsearch-native and elasticsearch-saml: Removes the current role mappings. All users are granted the owner role.
