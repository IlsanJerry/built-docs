IpFilterRulesets

The container for a set of IP filter rulesets.

Properties

rulesets (array[IpFilterRuleset], required)
List of rules

Example

{
   "rulesets" : [
      {
         "associations" : [
            {
               "entity_type" : "string",
               "id" : "string"
            }
         ],
         "description" : "string",
         "id" : "string",
         "name" : "string",
         "rules" : [
            {
               "description" : "string",
               "id" : "string",
               "source" : "string"
            }
         ],
         "total_associations" : 0
      }
   ]
}
