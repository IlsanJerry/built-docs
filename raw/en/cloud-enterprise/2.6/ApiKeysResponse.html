ApiKeysResponse

The response model for the API keys.

Properties

keys (array[ApiKeyResponse], required)
The list of API keys.

Example

{
   "keys" : [
      {
         "creation_date" : "2019-01-01T00:00:00Z",
         "description" : "string",
         "id" : "string",
         "key" : "string",
         "user_id" : "string"
      }
   ]
}
