Elastic Stack 6.7.0 released

Elastic Cloud Enterprise now supports the latest version of the Elastic Stack, which includes Elasticsearch and Kibana 6.7.0.

For more information about the features in Elasticsearch and Kibana 6.7.0, see:

  • Elasticsearch 6.7.0 Released
  • Elasticsearch 6.7.0 Release Notes
  • Kibana 6.7.0 Release Notes
