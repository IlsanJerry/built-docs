Getting started

There are two ways you can get Elastic Cloud Enterprise up and running:

  • Just want to kick the tires? Try the Quick Start Guide. There’s no faster way and even just a single virtual machine will get you started.
  • Planning a production system? Start with our Playbook for Production and go through the full set of the steps, from planning to deployment. You end up with a highly available installation that is ready for production. We built our playbook around several example scenarios that cover small, medium, and large deployments.
