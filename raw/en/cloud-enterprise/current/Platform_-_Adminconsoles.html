Get adminconsoles

Retrieves an overview of all adminconsoles.

Request

GET /api/v1/platform/infrastructure/adminconsoles

Responses

200

(AdminconsolesOverview) An overview of all adminconsoles.

Headers

x-cloud-resource-created (string)
The date-time when the resource was created (ISO format relative to UTC)
x-cloud-resource-last-modified (string)
The date-time when the resource was last modified (ISO format relative to UTC)
x-cloud-resource-version (string)
The resource version, which is used to avoid update conflicts with concurrent operations

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XGET https://{{hostname}}/api/v1/platform/infrastructure/adminconsoles \
-H "Authorization: ApiKey $CLOUD_API_KEY"

Delete adminconsole logging settings

Reset the default logging settings for this adminconsole instance.

Request

DELETE /api/v1/platform/infrastructure/adminconsoles/{adminconsole_id}/logging_settings

Path parameters

NameTypeRequiredDescription

adminconsole_id

string

Y

The identifier for the adminconsole instance

Responses

200

(LoggingSettings) The updated logging settings for the adminconsole instance

Headers

x-cloud-resource-created (string)
The date-time when the resource was created (ISO format relative to UTC)
x-cloud-resource-last-modified (string)
The date-time when the resource was last modified (ISO format relative to UTC)
x-cloud-resource-version (string)
The resource version, which is used to avoid update conflicts with concurrent operations
404

(BasicFailedReply) The logging settings for this adminconsole were not found. (code: adminconsoles.logging_settings.not_found)

Headers

x-cloud-error-codes (string; allowed values: [adminconsoles.logging_settings.not_found])
The error codes associated with the response

To perform this operation, you must be authenticated by means of one of the following methods: apiKey, basicAuth.

Request example

curl -XDELETE https://{{hostname}}/api/v1/platform/infrastructure/adminconsoles/{adminconsole_id}/logging_settings \
-H "Authorization: ApiKey $CLOUD_API_KEY"
