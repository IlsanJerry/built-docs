Install ECE on a public cloudedit

You can install ECE in many different ways. Choose the path that best fits your use case.

  1. Select the cloud provider:

    • Amazon Machine Images (AMI)
    • Google Cloud Platform (GCP)
    • Microsoft Azure

  2. Choose the operating system:

    • Ubuntu 16.04 LTS (Xenial Xerus)
    • Red Hat Enterprise Linux (RHEL) 7 and CentOS 7
    • SUSE Linux Enterprise Server (SLES) 12

  3. Identify the deployment scenario:

    • Small installation: For development, test, and small-scale use cases.
    • Medium installation: For many production setups.
    • Large installation: For deployments with significant overall search and indexing throughput.
