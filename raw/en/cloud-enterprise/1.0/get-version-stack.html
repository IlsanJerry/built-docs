Get version stack

Retrieves a single version stack configuration.

Request

GET /api/v1/stack/versions/{version}

Path Parameters

version (required)
(string) Stack version

Responses

200
(StackVersionConfig) Stack version was found
404
(BasicFailedReply) Stack version was not found
