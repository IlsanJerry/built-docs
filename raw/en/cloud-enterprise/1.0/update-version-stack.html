Update version stack

Updates the JSON configuration for an existing version stack.

Request

PUT /api/v1/stack/versions/{version}

Path Parameters

version (required)
(string) Stack version

Request body

(StackVersionConfigPost) (required) Stack configuration

Responses

200
(StackVersionConfig) Stack version was updated successfully
404
(BasicFailedReply) Stack version was not found
