Get TLS certificate

Retrieves a certificate in the TLS certificate chain.

Request

GET /api/v1/platform/configuration/security/tls/{service_name}

Path Parameters

service_name (required)
(string; allowed values: [adminconsole, proxy, ui]) The service certificate chain to read, one of adminconsole, proxy, ui

Responses

200
(TlsPublicCertChain) The TLS certificate chain, with private key omitted, for the given service and whether it was user specified
449
(BasicFailedReply) When running as an administrator (other than root), sudo is required (code: 'root.needs_sudo')
