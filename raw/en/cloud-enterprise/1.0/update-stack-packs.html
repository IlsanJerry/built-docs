Update stack packs

Bulk updates version stack configurations and templates using a configuration archive. Besides multipart/form-data requests, the endpoint supports application/zip and application/octet-stream requests with a binary body. Maximum size of the payload is 1Mb. If the archive contains a stack configuration that is already available through the API, then the existing configuration and template will be overwritten.

Request

POST /api/v1/stack/versions

Form Parameters

file (required)
(file) Zip file that contains one or multiple stack configurations

Responses

200
(StackVersionArchiveProcessingResult) File was processed successfully
400
(BasicFailedReply) Bad archive or something unexpected happened
