Stop a deployment

To stop a deployment in Elastic Cloud Enterprise:

  1. Log into the Cloud UI.

  2. On the Deployments page, select your deployment.

    Narrow the list by name, ID, or choose from several other filters. To further define the list, use a combination of filters.

  3. In the Deployment Management section, click Terminate deployment.

Elastic Cloud Enterprise will downscale the deployment before stopping it, but you can still restart the deployment with the same configuration later.

+ TIP: Without a current snapshot, only the configuration settings are kept. If there is a snapshot repository associated with the Elasticsearch cluster, you can restart the cluster with the same indices later.
