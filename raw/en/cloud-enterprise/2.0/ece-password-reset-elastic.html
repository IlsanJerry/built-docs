Reset the Password for the elastic User

For version 5.0 and later: You might need to reset the password for the elastic superuser if you cannot authenticate with the elastic user ID and are effectively locked out from a cluster.

To reset the password:

  1. Log into the Cloud UI.

  2. From the Deployments page, select your deployment.

    Narrow the list of the deployments by name, ID, or choose from several other filters. Use a combination of them to further define the list.

  3. From your deployment menu, select Security.
  4. Click Reset password.
  5. Copy down the auto-generated password for the elastic user.

The password is hashed after you leave this pane, so if you lose it, you need to reset the password again.

Reset the Password for the admin User

You might need to reset the password for the admin user if you cannot authenticate with the admin user ID and are effectively locked out from your entire installation.

If this happens, you’ll need to retrieve it or reset it.
