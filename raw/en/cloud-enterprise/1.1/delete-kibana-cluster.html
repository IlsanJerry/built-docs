Delete cluster

Deletes a Kibana cluster. Requires that you have already successfully issued _shutdown command against the cluster.

Request

DELETE /api/v1/clusters/kibana/{cluster_id}

Path Parameters

cluster_id (required)
(string) Identifier for the Kibana cluster

Responses

200
(EmptyResponse) The cluster has been successfully deleted
404
(BasicFailedReply) The cluster specified by {cluster_id} cannot be found (code: 'clusters.cluster_not_found')
412
(BasicFailedReply) The Kibana cluster has not been shutdown yet (code: 'clusters.cluster_plan_state_error')
449
(BasicFailedReply) When running as an administrator (other than root), sudo is required (code: 'root.needs_sudo')
