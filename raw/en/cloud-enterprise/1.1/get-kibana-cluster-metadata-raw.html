Get cluster metadata

Advanced use only: Retrieves the internal cluster metadata for a Kibana cluster as free-form JSON.

Request

GET /api/v1/clusters/kibana/{cluster_id}/metadata/raw

Path Parameters

cluster_id (required)
(string) Identifier for the Kibana cluster

Responses

200
(object) The cluster metadata was successfully returned
404
(BasicFailedReply) The cluster specified by {cluster_id} cannot be found (code: 'clusters.cluster_not_found')
