Get allocator metadata

Retrieves the metadata for a given allocator.

Request

GET /api/v1/platform/infrastructure/allocators/{allocator_id}/metadata

Path Parameters

allocator_id (required)
(string) The identifier for the allocator.

Responses

200

(array[AllocatorMetadataItem]) The allocator metadata was successfully returned.

404

(BasicFailedReply) The allocator specified by {allocator_id} cannot be found (code: 'allocators.allocator_not_found')
