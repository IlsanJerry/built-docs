Get all available Elastic Stack versions

By default it returns only available versions. Use parameter show_deleted to get all versions

Request

GET /api/v1/stack/versions

Query Parameters

show_deleted
(boolean) Whether to show deleted stack versions or not

Responses

200

(StackVersionConfigs) List of all available stack configurations with template version and structure
