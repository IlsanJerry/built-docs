Stop a Cluster

To stop a cluster in Elastic Cloud Enterprise:

  1. Log into the Cloud UI.
  2. Click on a cluster name from the Clusters panel and click Manage.
  3. In the Danger Zone section, click Stop cluster. Let the user interface guide you through the steps for stopping a cluster.

Elastic Cloud Enterprise will downscale the cluster before stopping it, but you can still restart the cluster with the same configuration later.

+ TIP: Without a current snapshot, only the configuration settings are kept. If there is a snapshot repository associated with the cluster, you can restart the cluster with the same indices later.
