Get an Elastic Stack version

Retrieves a single Elastic Stack version with template.

Request

GET /api/v1/stack/versions/{version}

Path Parameters

version (required)
(string) An Elastic Stack version

Responses

200
(StackVersionConfig) The Elastic Stack version was found
404
(BasicFailedReply) The Elastic Stack version was not found
