Get snapshot repositoriesedit

Retrieves a list of all available snapshot repository configurations. Privileged access required for configuration contents.

Requestedit

GET /api/v1/platform/configuration/snapshots/repositories

Responsesedit

200

(RepositoryConfigs) List of snapshot repository configs
